Aurelia - Hello World
===================
A simple Hello World starter project for Aurelia. (With bundling tasks)


Getting Started
-------------
You must have Node.js installed

The from the command prompt you must install jspm globally
> - npm install jspm@0.16.19 -g

Now from the command line enter the folder containing the package.json file and run the following:
> - npm install
> - jspm install
> - gulp watch

Application should now be viewable in browser at http://localhost:9000


Deploy to production
-------------
To deploy to production:
> - gulp deploy


The production version of the client files will be in the ./wwwprod folder.
